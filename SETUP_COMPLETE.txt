【请求流程 - 前端 API 调用】

其他电脑浏览器
        ↓ 访问 https://xxxx-xxxx-xxxx.ngrok-free.dev
ngrok 隧道（自动双向转发）
        ↓ 转换为 http://localhost:5173
你的电脑前端 Vite 服务器
        ↓ 识别到 /api 请求
代理转发（自动）
        ↓ 转发到 http://localhost:8000
你的电脑后端 FastAPI
        ↓ 调用数据库、LLM 等
响应数据
        ↓ 最终返回给前端
前端渲染
        ↓ 其他电脑浏览器显示结果

【排故调试】

如果前端无法调用 API：
  1. 检查后端是否运行：http://localhost:8000/
  2. 检查 ngrok 是否正常：http://127.0.0.1:4040
  3. 检查浏览器控制台是否有错误
  4. 确保 .env 中 CORS 配置正确

如果其他电脑看不到网页：
  1. 确认使用的是 ngrok URL，不是 localhost
  2. 检查防火墙是否阻止 ngrok
  3. 重启 ngrok 获取新 URL
  4. 确保 npm run dev 已在前端终端运行
